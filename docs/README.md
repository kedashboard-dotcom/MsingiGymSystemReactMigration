# Msingi Gym System

## Development
1. Run `.\scripts\setup.ps1`
2. Update .env files
3. Run `.\scripts\dev-start.ps1`

## Deployment
1. Update production .env files
2. Run `.\deployment\deploy.sh`

## API Endpoints
- POST /api/members/register
- POST /api/members/renew  
- GET /api/members/status
- POST /api/check-mpesa
